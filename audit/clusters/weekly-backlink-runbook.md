# WEEKLY BACKLINK RUNBOOK

Execute every Monday. 45-60 minutes total. Can be done by Brooks or handed to an assistant.

---

## Monday Checklist

### 1. Check Google Alerts (10 min)

- [ ] Open Google Alerts email digest
- [ ] For each new mention of "Hardlife Apparel," "HRDLF," "HRDLFcoin," or "Brooks Duvall":
  - Does the mention include a link to hrdlf.com, hardlifeapparelco.com, or hrdlfcoin.com?
  - If YES: log it in the tracking sheet as a "linked mention" — no action needed
  - If NO: send Template A (warm thank-you) from brand-mention-template.md within 48 hours
- [ ] Log all new mentions in the tracking sheet (see below)

### 2. Check Source Platforms (15 min)

- [ ] Log into Qwoted — check for new queries matching your 10 beats
- [ ] Log into Terkel — check for new prompts
- [ ] Log into Featured.com — check for new expert roundups
- [ ] Check X for #JournoRequest #PRRequest — search for: streetwear, fashion, AI, founder, skate, crypto, Philadelphia
- [ ] Respond to 3-5 relevant queries using the journalist bio response template
- [ ] Log responses in the tracking sheet

### 3. Broken Link Scan (10 min)

- [ ] Run one search query from the broken-link list:
  - Week 1: `"best streetwear brands" "crooks and castles"`
  - Week 2: `"indie streetwear" "diamond supply"`
  - Week 3: `"underground brands" "10.deep"`
  - Week 4: `"streetwear brands to watch 2024"`
  - Rotate monthly
- [ ] Click through top 5 results — check for dead outbound links
- [ ] If dead link found: send broken-link outreach email (template in broken-link-emails/)
- [ ] Log opportunity in tracking sheet

### 4. Competitor Backlink Outreach (10 min)

- [ ] Pick 2 targets from competitor-backlink-targets.csv (work top-to-bottom by priority)
- [ ] Personalize the outreach email from outreach-emails/ folder
- [ ] Send emails
- [ ] Update outreach_status in CSV: "pending" → "sent [date]"
- [ ] After 7 days with no response: send one follow-up (see follow-up template below)

### 5. Update Tracking Sheet (5 min)

- [ ] Log all actions taken this week
- [ ] Update status of pending outreach (sent → responded / no response / link earned)
- [ ] Count total new links earned this month
- [ ] Note any new opportunities discovered

---

## Follow-Up Template (7 days after initial outreach, no response)

```
Subject: Re: [original subject]

Hey [name],

Following up on my note from last week about [topic].

No pressure — just wanted to make sure it didn't get
buried. Happy to provide any additional details.

Brooks
hrdlf.com
```

One follow-up only. If no response after follow-up, move to next target. Do not send a third email.

---

## Monthly Review (first Monday of each month)

- [ ] Count total links earned vs. outreach sent (target: 5-10% conversion rate)
- [ ] Identify which outreach type performs best (mention recovery vs. broken link vs. direct pitch)
- [ ] Update competitor-backlink-targets.csv with new targets discovered
- [ ] Add 5 new broken-link search queries based on recent SERP changes
- [ ] Review Google Alerts — add new alert queries if brand awareness is growing

---

## Key Metrics to Track

| Metric | Target (Month 1) | Target (Month 3) | Target (Month 6) |
|--------|------------------|-------------------|-------------------|
| Outreach emails sent | 20 | 40 | 40 |
| Links earned | 2-3 | 5-8 | 8-12 |
| Source platform responses | 12-15 | 20 | 20 |
| Source platform pickups | 1-2 | 3-5 | 5-8 |
| Unlinked mentions recovered | 1-2 | 2-3 | 3-5 |
| Estimated DA growth | 10-15 → 15-18 | 18-22 | 22-30 |

---

*Consistency beats volume. 20 personalized emails per month > 100 spray-and-pray pitches. Every outreach email must be in Brooks's voice — direct, factual, no hype.*
